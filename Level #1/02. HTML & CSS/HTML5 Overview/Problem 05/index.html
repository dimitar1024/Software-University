Headings and Paragraphs

  • Headings Tags: < h1 > - < h6 >

    • <h1>Heading 1</h1>
    <h2>Sub heading 2</h2>
    <h3>Sub heading 3</h3>

  • Paragraph Tags: < p >

    • <p>This is my first paragraph</p>
    <p>This is my second paragraph</p>

  • Section Tags: < span > - < div >

    • <div style="background: skyblue;">
    This is a div
    </div>
